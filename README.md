# clj-tictactoe

A simple tictactoe program

## Installation

```
$ git clone https://github.com/timothyhahn/clj-tictactoe.git

$ cd clj-tictactoe/

$ lein deps

$ lein compile
```

## Usage

You can run clj-tictactoe's jar with:

```
$ java -jar clj-tictactoe-1.0.1-standalone.jar
```
